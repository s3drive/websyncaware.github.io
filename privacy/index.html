Privacy policy

This Privacy Policy describes how your personal information is collected, used, and shared when you use any software program provided by the Sync Solutions (the "Company") downloaded by You on any electronic device (the "Application").


The Application doesn't collect or transmit data beyond your media files which are synchronized with your personal S3 compatible account, except:

- if QR code authorization (optional) is used, this Application will pass the encrypted credentials over "Pusher" service back to web client which rendered the QR code.

- if "Open web client" feature (optional) is used, this Application will pass one-time encryption key over "Pusher" service back to web client which will then decrypt the encrypted credentials passed in the URI.

- in case of an Application error, this Application may send an error report using "Slack" Webhook API. This allows us to address any potential problems you might be having with this Application. We're logging error message, stack trace, as well as information about your phone model, system and Application version that you're using.
You can disable this feature in the Settings screen.

- if user decides to use built-in debug log export function, the Application logs will be uploaded to "Filestack.com". User has a possibility to review the Application logs before uploading.
Uploaded files will be deleted after a period of one month or upon the user request.

- if user buys a paid feature there is a connection made to Google Billing API, so the order can be fulfilled. When Application starts, there is a connection made to Google Billing API to verify the validity of the purchase and auto-enable paid features.

Your media files are uploaded to your personal S3 compatible account.
The Company has no control over, and assumes no responsibility for, the content, privacy policies, or practices of your S3 provider.
We strongly advise You to read the terms and conditions and privacy policies of your respective S3 provider.

We're not logging your IP address, device serial number, MAC address, Wi-Fi name or your location.
